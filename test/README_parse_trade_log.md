# 交易日志解析脚本使用说明

## 功能
自动解析交易日志文件，提取调仓信息并按日期生成CSV报告。

## 使用方法

### 1. 处理今天的日志
```bash
python parse_trade_log.py
```

### 2. 处理指定日期的日志
```bash
python parse_trade_log.py 20251017
```

### 3. 交互式选择文件
如果没有找到指定日期的日志文件，脚本会：
- 显示所有可用的日志文件
- 询问是否处理最新的文件

## 输出文件
脚本会在 `trade_reports/` 目录下生成以下CSV文件：

- `positions_YYYYMMDD.csv` - 仓位汇总（股票代码和净持仓）
- `buy_trades_YYYYMMDD.csv` - 买入明细（股票代码、数量、价格、金额、时间）
- `sell_trades_YYYYMMDD.csv` - 卖出明细（股票代码、数量、价格、金额、时间）

## 日志文件格式要求
- 文件名格式：`ta_YYYYMMDD_HHMMSS`
- 调仓记录格式：`股票代码|数量变化|日期|时间|v1|v2|价格`
- 示例：`603395.SH|100|140109|143100|0|0|33.835`

## 统计信息
脚本会输出当日交易统计：
- 总交易笔数
- 买入/卖出笔数
- 买入/卖出金额
- 净流入金额
- 持仓股票数
